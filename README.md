# 🌌 Star Wars Character Search

A mini-project that allows users to search for **Star Wars characters** using the [SWAPI](https://swapi.py4e.com/) API.  
The app fetches details such as birth year, gender, eye color, hair color, skin color, homeworld, starships, and vehicles, and displays them in styled character cards with images.

---

## 🚀 Features
- Search for Star Wars characters by name.
- Display character details in a clean card layout.
- Show character images (with fallback placeholder if unavailable).
- Fetch and display homeworld, starships, and vehicles dynamically.
- Responsive and visually appealing UI with CSS styling.

---

## 🛠️ Tech Stack
- **HTML** – Structure of the app  
- **CSS** – Styling and layout (custom theme with background and card design)  
- **JavaScript (ES6)** – Fetching data from SWAPI and rendering character cards  
- **SWAPI API** – Source of Star Wars character data  

---

## 📂 Project Structure
- index.html        # Main HTML file
- style.css         # Styling for the app
-  script.js         # JavaScript logic (API calls, rendering)
- OIP.jpeg          # Background image

---

## ⚡ How It Works
1. Enter a character name in the search box.  
2. Click the **Search** button.  
3. The app fetches data from SWAPI and displays character cards with details and images.  
4. If no characters are found, a message is shown.
